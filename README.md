## 📋 Included Tools

| Script                        | Category              | Description |
|------------------------------|-----------------------|-----------|
| `file_organizer.py`          | File Management      | Automatically sorts files into folders by type, date, or name |
| `bulk_renamer.py`            | File Management      | Rename multiple files with custom patterns or numbering |
| `password_generator.py`      | Security             | Strong random password generator with customization |
| `web_scraper.py`             | Web Automation       | Simple news, price tracker, or data collector (with rate limiting) |
| `qr_code_generator.py`       | Utilities            | Generate and save QR codes from text/URLs |
| `unit_converter.py`          | Utilities            | Convert between units (length, weight, temperature, etc.) |
| `backup_tool.py`             | Backup               | Backup folders with timestamped archives |
| `email_sender.py`            | Communication        | Send automated emails (with attachments support) |
| `weather_cli.py`             | API Integration      | Get current weather + forecast from command line |
| `todo_manager.py`            | Productivity         | Simple command-line todo list with persistence |
